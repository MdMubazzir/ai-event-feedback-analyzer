# AI Event Feedback Analyzer

This project is an AI-powered web application that analyzes participant feedback for events using a transformer-based NLP model.

## Features
- Submit feedback for events
- AI-based sentiment analysis using BERT
- Admin dashboard with analytics
- Top event ranking based on feedback
- Flask backend with dynamic frontend

## Tech Stack
- Python (Flask)
- Hugging Face Transformers
- PyTorch
- HTML, CSS, JavaScript

## How to Run
```bash
pip install -r requirements.txt
python app.py
